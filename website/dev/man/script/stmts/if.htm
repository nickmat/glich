Statement: if

An if statement allows one or more statements to be run, depending on a given condition.

The statement may include zero or more 'elseif condition' sections and an optional 'else' section. The statement must end with a semicolon.

if Statement Format
if bool_expr { statements ... } elseif bool_expr { statements ... } else { statements ... };
Example
if value = 10 { write "Value is a round 10" nl; } elseif value < 5 { write "Value as less than 5" nl; } elseif value = 100 { write "Value is exactly 100" nl; } else { write "Value is boring" nl; };
