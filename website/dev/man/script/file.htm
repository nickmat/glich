Statement: file

The file statement opens and prepares a text file, ready to read and write from.

file Statement
FormatExample
file filecode filename switch; file note "note.txt" append;

The statement defines a filecode which can later be used as a qualifier to the write statement and @read function.

The filename is expected in the form of a string expression.

The switch is optional, if given it can be one of the following names:

SwitchNote
read Open an existing file for reading text.
write Open for writing, truncating any existing text.
append Open for writing, adding new text to the end.

If no switch is given then 'read' is assumed.

Example Script
let filename = "Notes-file.txt"; file note filename write; write.note "This is a test string." nl; write.note "And this is a second." nl; file note2 filename read; write @read.note2 nl; write @read.note2;
Output:-
This is a test string. And this is a second.
