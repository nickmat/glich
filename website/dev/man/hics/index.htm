HistoryCal Script Language Extension hics

This extension adds the ability to define and manipulate a number of calendar schemes.

It adds four statements: scheme, grammar, format and lexicon to define a calendar scheme. It also adds three operators: text, date and record (referred to as casts), to convert between different representation of a date and a fourth: element, to convert between representation of an element of a date.

The context setting can be used to set the literal field values of +infinity and -infinity to future and past respectively.
