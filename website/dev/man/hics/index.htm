Glich Script Language Extension Hics

This extension adds the ability to define and manipulate a number of calendar schemes.

It adds four statements: scheme, grammar, format and lexicon to define a calendar scheme. It also adds a number of functions: @text, @date, @record and @phrase to convert between different representation of a date and another: @element, to convert between representation of an element of a date.

The context setting hics can be used to set the literal field values of +infinity and -infinity to future and past respectively and to default integer input to field types.
