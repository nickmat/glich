Introduction

Glich is an extendable script language. It is a self contained language in its own right, but its main purpose is to provide a base for more specialised applications.

Basic Glich

The basic language includes the language statements, all the value types, some built-in functions and a general purpose object type.

Hics Extension

Hics (HistoryCal Script) is the name given to the Calendar definitions, conversions and handling of date ranges.
